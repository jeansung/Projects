Wake Me Up When

A distance based alarm clock Android application. During August 2014, I had to take 2 buses to work for my second job. It was early and I wanted to sleep on the first bus. However, it did not get into the bus stop at a reliable time, so using a regular time based alarm clock was out of the question. However, because the bus stop was at a consistent location, a distance based alarm clock might be reasonable. I thus sought to learn the Google Maps API / basics of Android and created a proof of concept application.

Sum Plus Plus

I built a tool that allows a user to specify a custom environment for creating and checking the feasibility of constraints. This means that a user will input types and values, as well as a set of constraints, and then they will receive a sandbox Excel file to play around in, checking to see how various configurations do/do not follow the rules. The project notebook is where you can find updates on my project.

  • Download here: Project website
  • Date: Fall 2014
  • Affiliation: CS 111: Domain Specific Languages, Harvey Mudd College

Physics Sim

Traditional programming languages often separate the user and the programmer into two non intersecting camps when it comes to making tools. The programmer is the only one who can add new features and create with the tool. We explored the concept of backing a software tool with a programming language written in the domain expert's field in order to make a more reusable tool.
