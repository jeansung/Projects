Wake Me Up When

During the Summer, I had to take 2 buses to work for my second job. It was early and I wanted to sleep on the first bus. However, it did not get into the bus stop at a reliable time, so using a regular time based alarm clock was out of the question. However, because the bus stop was at a consistent location, a distance based alarm clock might be reasonable.

In progress right now.

Piconot

I worked with a partner to write a wrapper programming language for Picobot . Picobot is a programming language that specifies the state of a mini robot based on its conditions. The new syntax focuses on readability and reshifting the program into an "if then" paradigm, which is helpful for introductory CS students.

  • Fork or Clone: Github Repo
  • Date: Fall 2014
  • Affiliation: CS 111: Domain Specific Languages, Harvey Mudd College

Physics Sim

Traditional programming languages often separate the user and the programmer into two non intersecting camps when it comes to making tools. The programmer is the only one who can add new features and create with the tool. We explored the concept of backing a software tool with a programming language written in the domain expert's field in order to make a more reusable tool.

  • Fork or Clone: Github Repo
  • Date: Summer 2014
  • Affiliation: Department of Computer Science, Harvey Mudd College
